# Solutions: please read

The solutions are worked examples following the tutorial activities.

The are not examplars. That is, they are not perfect and not intended as model answers.

They focus on the particular concept being taught and so may ommit docstrings and error handling.

They are a single solution, intended at a beginner level. More efficient solutions may well exist. Feel free to contribute other solutions, use the [issues tab in GitHub](https://github.com/shiressl1/COMP0035-tutorials-2025/issues) and create a new issue title with 'Alternative solution to `<some activity number>`' whichever activity.

I am human, and I also make mistakes! You may find bugs and mistakes, if so please report them using the [issues tab in GitHub](https://github.com/shiressl1/COMP0035-tutorials-2025/issues).
